# BiliQueue v0.1.10 测试版

BiliQueue 是一个本地运行的 B 站直播横向排队工具。程序提供网页控制台和 OBS 浏览器横条，可通过弹幕管理排队、取消排队和礼物插队。

## v0.1.10 调整

### 顶部固定预览

OBS 地址、复制地址、单独打开和实时预览已移动到控制台顶部。

预览区域宽度与下方双栏 `.grid` 一致。页面向下滚动后，预览会固定在浏览器顶部。横条会同时按照预览区域的宽度和高度等比例缩放，不会按实际像素高度撑大控制台。

### 字体文件夹

程序目录新增：

```text
fonts/
```

支持以下格式：

```text
.ttf
.otf
.woff
.woff2
```

将字体文件复制到 `fonts` 后，在控制台点击“刷新字体列表”。当前区域、队列区域和说明区域可以分别选择字体。

测试包不附带字体文件。字体文件由使用者自行准备，并需确认相应授权范围。

### 文字透明度

当前区域、队列区域和说明区域分别增加文字透明度设置。

透明度控制方式与原背景设置一致：

- `0%`：完全透明
- `100%`：完全显示

### 队列排序

队列项目仍可从整行任意位置拖动。左侧新增拖动手柄图标，用于提示该项目支持拖动。

每个队列项目增加：

```text
↑ 上移一位
↓ 下移一位
```

当前正在进行的第一位用户也能通过拖动或“下移”调整位置。


### 头像大小

OBS 横条中的用户头像增加大小设置，影响当前区域、队列区域中的头像显示。当前区域头像位置已修正为与用户名组合一起居中显示。

### 全局渐变和三区背景

全局背景增加：

- 背景颜色
- 顶部透明度
- 底部透明度
- 渐变开始位置
- 渐变结束位置

顶部和底部设置不同数值时，会形成从上到下的透明度渐变。开始位置之前保持顶部透明度，结束位置之后保持底部透明度。

三个区域另外拥有独立的背景颜色和透明度：

- 当前区域背景
- 队列区域背景
- 说明区域背景

区域背景位于全局渐变上层，可以叠加不同色调。

旧版配置会自动迁移。原背景透明度会同时写入顶部和底部，因此升级后会先维持原来的纯色背景效果。

## 数据与目录

默认目录结构：

```text
BiliQueue-windows-amd64.exe
启动-BiliQueue.cmd
fonts/
data/
├── auth.json
├── config.json
├── queue-today.json
├── avatars/
└── backups/
```

`auth.json` 含 B 站登录 Cookie，请勿分享。

在同一目录覆盖新版 EXE，可以继续读取原登录状态和配置。解压到新目录时，需要复制旧版 `data` 文件夹；自定义字体还需要复制 `fonts` 文件夹。

## 使用方法

1. 双击 `启动-BiliQueue.cmd`。
2. 浏览器打开 `http://127.0.0.1:18303/control`。
3. 首次使用点击“扫码登录”。
4. 填写直播间号并连接。
5. OBS 添加浏览器来源，地址填写 `http://127.0.0.1:18303/overlay`。
6. OBS 宽度和高度按照顶部预览区域中的建议尺寸填写，FPS 设置为 30。

