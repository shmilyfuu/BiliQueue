## BiliQueue v0.1.6

首个 GitHub 发布版本。

### 主要功能

- B 站扫码登录与直播间弹幕连接
- 弹幕“排队”和“取消排队”指令
- 当前用户、等待队列和说明区域独立布局
- 单行、双行队列及滚动显示
- 礼物电池门槛插队与礼物优先区排序
- OBS 浏览器横条
- 当天队列异常退出恢复
- 配置导入、导出和自动备份

### v0.1.6 调整

- 修复普通弹幕排队用户头像无法获取的问题
- 从 `DANMU_MSG` 扩展用户信息读取头像
- 增加本机头像代理与 `data/avatars` 缓存
- 旧队列用户再次发送排队指令时可补充头像
- 增加配置导入和导出
- 导入配置前自动备份当前设置

### 下载

- `BiliQueue-v0.1.6-windows-test.zip`：Windows AMD64 测试包
- `BiliQueue-v0.1.6-source.zip`：完整源码

### 升级提示

需要保留登录状态和现有设置时，请保留旧版的 `data` 文件夹。`data/auth.json` 含 B 站登录 Cookie，请勿分享。
